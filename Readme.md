## Project Overview
This project delivers a comprehensive learning pathway for beginners to build games using **Godot Engine** and integrate blockchain features using **Cardano**.  
The program includes:
- 10–12 lessons,
- 10 videos,
- 10 articles,
- slide decks,
- exercises,
- demo mini-game,
- workshops, Game Jam, AMA,
- full documentation and open-source code.
---
## Objectives
- Provide the first structured Godot + Cardano education guide.
- Produce high-quality multimedia content (video, slides, articles).
- Help new learners with real example projects and exercises.
- Expand community engagement via workshops and Game Jam.
- Publish all outputs openly to maximize educational impact.
